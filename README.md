## 🎥 Video Conferencing App (React)
A real-time video conferencing web application built with React that enables users to join virtual rooms, communicate via video/audio, and collaborate seamlessly.
This project demonstrates real-time communication, peer-to-peer connections, and modern UI design.

---

## 📌 About the Project
This application allows multiple users to join a virtual meeting room and communicate using video and audio.

It was built to explore:
- Real-time communication using WebRTC
- Managing peer connections
- React state management
- Handling media streams (camera & microphone)

---

## ✨ Features
- 🎥 Real-time video calling
- 🎙️ Audio communication
- 👥 Multi-user room support
- 🔗 Unique meeting room links
- 📺 Live video streams
- ⚛️ Built with React functional components
